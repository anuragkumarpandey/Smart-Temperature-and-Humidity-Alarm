# Smart Temperature and Humidity Alarm System

Bachelor Informatik Project - Hardware + Software

## Features
- DHT11 Temperature/Humidity sensor
- I2C 16x2 LCD display  
- 2 Status LEDs (Green=Normal ≤25°C, Red=Hot >25°C)
- Buzzer alarm (>30°C)

## Hardware
- Arduino Uno
- DHT11 sensor
- I2C LCD (A4/A5)
- 2 LEDs with 220Ω resistors 
- Active buzzer
